# Astro

## Overview
Astro is an exciting spaceship asteroid shooter game where players pilot their spacecraft through an asteroid field, trying to destroy as many asteroids as possible while avoiding collisions.

## Gameplay
- Navigate your spaceship using the arrow keys.
- Press the spacebar to shoot at asteroids.
- Earn points for each asteroid destroyed.
- Avoid collisions with asteroids, as they will decrease your health.

## Features
- Challenging levels with increasing difficulty.
- Power-ups that enhance your spaceship's capabilities.
- High score tracking to challenge friends.

## Installation
To play Astro, clone the repository and open the main game file in your browser.

```bash
git clone https://github.com/Quantum-Youssef-Amr/orpita-task-tot-1-Solution.git
cd orpita-task-tot-1-Solution
```
